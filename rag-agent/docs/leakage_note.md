중복 이슈를 명시 기록한다.

## 1. 사용 임베딩 모델과 학습 코퍼스

| 모델 | dense 검색 사용 | HiTab 포함 가능성 | 비고 |
|---|---|---|---|
| `BAAI/bge-large-en-v1.5` | 예(주 baseline) | **불명/낮음** | 공개 학습 데이터에 HiTab(2022) 명시 없음. 일반 웹·MS MARCO 계열 중심 |
| `intfloat/multilingual-e5-large-instruct` | 예(대조군) | **불명** | 대규모 다국어 웹 코퍼스. HiTab 표가 통계청 공개 PDF 유래라 웹 중복 가능 |

## 2. 누수 경로 평가

- **질문↔표 정렬 라벨 누수**: 없음. gold_table_id는 원본 주석에서 기계적으로 가져오며, 임베딩 모델은
  라벨을 본 적 없다(비지도 인코더). 검색 정답은 모델 학습과 독립.
- **표 내용 사전학습 노출**: HiTab 표는 Statistics Canada 등 공개 통계표에서 유래 → 임베딩 모델의
  웹 코퍼스에 원문 일부가 포함됐을 가능성을 배제 못함. 단, 이는 BM25/dense 모두에 동일하게 적용되며
  특정 검색기에 편향을 주지 않음(상대 비교는 유효).
- **fine-tune 누수**: 두 임베딩 모델 모두 본 실험에서 **재학습/파인튜닝하지 않음**(offline 인코딩만).
  HiTab QA 라벨로의 적합화 경로 없음.

## 3. 통제

- 임베딩은 라벨 없이 offline 1회 인코딩 후 디스크 캐시(chroma). seed=42 고정.
- 동일 코퍼스·동일 쿼리 위에서 BM25(어휘) vs dense(임베딩)를 쌍대 비교 → 사전학습 노출이
  있더라도 두 방식에 공통 적용되어 **상대 우열 해석은 보존**된다.
- 절대 수치(R@k)는 사전학습 노출로 상향 편향 가능성을 남겨두며, 본 노트로 투명 공개한다.
